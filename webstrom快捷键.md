选择相同的字符串：Alt + j 

多选后使用：ctrl + shift + 右箭头

Alt + Shift + 上箭头：块移动(if(){},while(){}语句块的移动)

向上移动一行：Ctrl + Shift + 上箭头

向下移动一行：Alt + Shift + 下箭头 

召唤终端：Alt + F12

折叠全部代码：Ctrl + Shift + - 

 展开全部代码：Ctrl + Shift + +

最近打开的文件：Ctrl + E 

最近更改的代码：Alt + Shift + C 

向上增加一行：Ctrl + Alt + 回车 

向下增加一行  （其实直接enter就可以，只不过和上边区分）：Ctrl + Shift + 回车 

复制当前一行代码：Ctrl + D

删除当前一行代码：Ctrl + Y

格式化代码：Ctrl + Alt + L 

Ctrl + J： 自动代码

单行注释： Ctrl + / 

块注释：Ctrl + Shift + /

大小写转换：Ctrl + Shift + U 

多行编辑：按住鼠标滚轮 往下拉
