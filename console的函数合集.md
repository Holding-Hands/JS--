console的函数合集

```js
const a = {};
const arr [1,2,3];

console.error('错误'); // 控制台打印出红色的报错
console.dir(a); // 打印出详细结构包括原型
console.table(arr); // 以表格形式打印数组
​```
console.time('zcy');
console.timeEnd('zcy'); // 这两行代码一起使用，记录执行的时间，参数需要相同当作标记符号

​```
console.warn(); // 打印出警告消息
```
