```js
解决方案
1. 跨页选择

2. 下拉加载，之前选择的数据消失

但上面方法看似完美，我们忽略了一个问题，就是全选的时候，我们需要多加一个判断
好了这下就解决啦

/**
     * 选择表格 列表 触发的事件
     * @param selectedRowKeys
     */
    onSelectChange (selectedRowKeys, selectedRows) {
      if (this.selectedRowKeys.length <= selectedRowKeys.length + 1 && this.selectedRowKeys.length !== 1) {
        this.selectedRowKeys = selectedRowKeys
        this.aTableSelectedRows = selectedRows
      }
      this.$emit('updateSelectedData', this.aTableSelectedRows)
    },
/**
     * 点击【全选】
     */
    onSelectAll (selectedRowKeys, selectedRows) {
      this.selectedRowKeys = selectedRowKeys
      this.aTableSelectedRows = selectedRows
    },
```

### 4. 项目优化

* 使用路由懒加载
* 事件的销毁（beforeDestory）
* 图片懒加载
* 第三方插件按需引入（babel-plugin-component）以达到减小项目体积的目的
* 长列表优化，使用虚拟滚动vue-virtual-scroll-list
* 防抖节流实现（按钮点击，搜索，页面滚动等情况）
* 浏览器缓存，为了提高用户加载页面的速度，对静态资源进行缓存是非常必要的
* CDN 的使用，浏览器从服务器上下载 CSS、js 和图片等文件时都要和服务器连接，而大部分服务器的带宽有限，如果超过限制，网页就半天反应不过来。而 CDN 可以通过不同的域名来加载文件，从而使下载文件的并发连接数大大增加，且CDN 具有更好的可用性，更低的网络延迟和丢包率 。





### 5. geoJSON相关

* GeoJSON是一种对各种地理数据结构进行编码的格式。GeoJSON对象可以表示几何、特征或者特征集合。GeoJSON支持下面几何类型：点、线、面、多点、多线、多面和几何集合。GeoJSON里的特征包含一个几何对象和其他属性，特征集合表示一系列特征。

* [geoJSON](https://www.oschina.net/translate/geojson-spec#polygon)特征集合
  * type 必须的 指的是Feature（特征）还是FeatureCollection（特征集合）
  * geometry 必须的 图形
    * coordinates 坐标
    * type 类型 点、线、面、多点、多线、多面和几何集合
    * 单独的geometry 也是标准的geoJSON格式，能被[geoJSONio](https://geojson.io/)识别
  * properties 属性 非必须

```json
{ "type": "FeatureCollection",
  "features": [
    { "type": "Feature",
      "geometry": {"type": "Point", "coordinates": [102.0, 0.5]},
      "properties": {"prop0": "value0"}
      },
    { "type": "Feature",
      "geometry": {
        "type": "LineString",
        "coordinates": [
          [102.0, 0.0], [103.0, 1.0], [104.0, 0.0], [105.0, 1.0]
          ]
        },
      "properties": {
        "prop0": "value0",
        "prop1": 0.0
        }
      },
    { "type": "Feature",
       "geometry": {
         "type": "Polygon",

           [ [100.0, 0.0], [101.0, 0.0], [101.0, 1.0],

     ]
```
